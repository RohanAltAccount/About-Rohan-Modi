# What it is
This is my personal webpage and portfolio. It is a static site that originally was built with classic HTML/CSS, but I learned React and migrated it. It uses Tailwind CSS and Astro for deployment and hosting. Feel free to check it out! (https://rohanaltaccount.github.io/About-Rohan-Modi/). Building this website improved my ability to work with different frameworks as a developer. Additionally, it increased my frontend-related knowledge. I plan on making it look more sleek and appealing. I have a list of my achievements, which are being constantly updated. Basically, this is my LinkedIn page since I don't meet the age requirement yet. I want to organize these accomplishments into nice little hoverboxes, so I plan on implementing that in the near future.

P.S. Try it on Dark Mode. Trust me, it looks way cooler. (It saves your setting every time you open it. I don't know why you would be doing that multiple times, but that's there for convenience.)
